ce projet est un site web utuliser pour prendre un rendez-vous de vaccin contre le COVID-19 . ce site se décompose en 3 pages . Une page de signup qui contient un formulaire à remplir avec une validation pour le mot de passe et le numero de carte d'identité puis le données seront stockées dans le localstorage , lorsque le signup se fait avec succes nous allons se deplacé à une autre page de signin et aprés faire le signin nous allons passer automatiquement à la page principales pour trouver nos donneés qui sont traitées avec le DOM et la date prevus de vaccin
